# What the fuck is this?
기숙사 자가진단 매크로 프로그램입니다.

이 프로그램을 사용하기 위해서는 사용자 데이터를 담아두는 'user.csv' 파일이 필요합니다.

<br>

# How To Use
1️⃣ user.csv에 사용자 정보를 등록합니다.

2️⃣ '작업 스케줄러'를 통해서 매번 특정 시간에 작동하도록 설정합니다.
